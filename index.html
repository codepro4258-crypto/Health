Open Patient Dashboard

Care coordination for patients and guardians

This project provides a lightweight monitoring flow where patients log routines and vitals, while guardians can monitor progress and receive alerts when risk thresholds are exceeded.

Run App (Server route) Guardian View

Demo users: Patient ID u1 and Guardian ID g1.

Patient workflow

  • View daily routines and adherence progress.
  • Mark routine tasks as completed.
  • Log blood pressure and pulse measurements.

Guardian workflow

  • Monitor linked patient adherence and latest vitals.
  • Receive real-time updates via SSE events.
  • Access alerts and acknowledge/resolution flows.

Risk and safety checks

  • Critical blood pressure threshold: 140/90 or higher.
  • Pulse warning threshold: 120 bpm or higher.
  • Audit log events for guardian monitor access.
Built for the Health repository MVP. Start the server and open http://localhost:3000.
